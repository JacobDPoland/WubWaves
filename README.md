# WubWaves
**Wavey and circular audio visualizer with dynamic coloring based on emotional sentiment analysis**

### Technologies:
* Front-end: HTML/CSS/JS
* Back-end: Node.js, YTDL-core module, TheAudioDB API, and Google's Natural Language API
